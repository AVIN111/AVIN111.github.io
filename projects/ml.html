Smart Navigation web app

  • Tech Stack: Django, Python, Javascript, AWS(RDS), HTML, CSS, bootstrap, HERE map API, yolov5
  • Website URL: Link
  • Github URL: Project Link

The web app created for smart and user friendly navigation in outdoor and indoor locations. User can log in or SignUp for the web app, each user will be given a unique ID so that admin can track the location of the users for security purpose, Outdoor avigation will done using Geofencing annd GPS technique and as for indoor navigaton it will be done using IOT devices such as ESP32 based on bluetooth or wifi communiation. The web app will guide user to different location displaying information whenever user comes in contact of a geofence. For more easy navigation the app will be provided with object detection functionality with which user can click the picture of the area or particular item to know about it.
